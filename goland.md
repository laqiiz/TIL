## Shortcut

The order I use frequently.

* Formatter:  Ctl + Alt + L

* Move line: Shift + Alt + Up/Down

* Run: Alt + Shift + F10

* Embed variable: Ctl + Alt + N
  * Extract variable: Ctl + Alt + V
